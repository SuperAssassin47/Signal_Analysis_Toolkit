# Scientific_Signal_Analysis_Toolkit
The Scientific Signal Analysis Toolkit (or S.S.A.T) is a collection signal analysis engines which allow the user to view alerts in real-time.

# Current abilities

v1.0.0 brings you the ability to passively listen for signals both on Earth and on space. The first engine, the multi-sensor correlation engine, listens for seismic spikes (such as earthquakes) and alerts you if they occur. The second engine, the multi-sensor space weather engine, passively listens space weather such as solar flares and solar wind shocks. This also alerts you if either kne occurs. But the common element of them both is that both engines will show a correlation alert if both singular alerts appear at the same time.

# Future optimisations

In future updates, I will aim to introduce more signal analysis engines to enhance the overall UX.
